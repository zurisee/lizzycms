Readme: code/
=============

Note: remove leading '#' to activate a file.

Sample Files:
-------------
#@daily-task.php				: code to be executed once per day
#@sample-service-task.php		: code to be invoked from other process
#@scheduled-task-sample.php		: code to be executed based on a schedule (triggered by external process, e.g. cron)
#init-code.php					: code executed whenever a page is requested
#user-macro-min-template.php	: macro skeleton
#user-macro-template.php		: macro template with explanations
#var-definitions.php			: example of variable definition code
